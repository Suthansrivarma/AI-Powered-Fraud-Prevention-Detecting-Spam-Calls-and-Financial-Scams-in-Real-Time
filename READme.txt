AI-Powered Fraud Detection System
A system to detect and prevent spam calls and financial fraud using AI and NLP. 
It provides real-time alerts to users for potential fraud based on call metadata and financial transaction patterns.

Features
Spam Call Detection: Detects suspicious calls using call metadata and NLP analysis.
Financial Fraud Detection: Identifies fraud in financial transactions through pattern analysis.
Real-Time Alerts: Blocks or alerts users when fraud is detected.


Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Python
NLP: NLTK
